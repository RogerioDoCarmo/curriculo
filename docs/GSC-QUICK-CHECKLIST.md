# Google Search Console - Quick Checklist

## 🎯 Quick Start (5 Minutes Per Domain)

### For Each Domain:

#### 1. Add Property

- [ ] Go to [Google Search Console](https://search.google.com/search-console)
- [ ] Click "Add Property"
- [ ] Choose "URL prefix"
- [ ] Enter: `https://rogeriodocarmo.com` (or other domain)
- [ ] Click "Continue"

#### 2. Verify Ownership (Choose One Method)

**Option A: HTML File (Easiest)**

- [ ] Download verification file from Google
- [ ] Add to `public/` folder
- [ ] Deploy to production
- [ ] Click "Verify"

**Option B: Meta Tag**

- [ ] Copy meta tag from Google
- [ ] Add to `app/[locale]/layout.tsx` metadata
- [ ] Deploy to production
- [ ] Click "Verify"

#### 3. Submit Sitemap

- [ ] Go to "Sitemaps" in left sidebar
- [ ] Enter: `sitemap.xml`
- [ ] Click "Submit"
- [ ] Wait for "Success" status
