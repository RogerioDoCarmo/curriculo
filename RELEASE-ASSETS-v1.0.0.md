## 🚀 Deployment Options

### Option 1: Vercel (Recommended)

Already deployed automatically to https://rogeriodocarmo.com

### Option 2: Static Hosting

Extract the archive and upload to any static hosting:

- Netlify
- GitHub Pages
- AWS S3 + CloudFront
- Cloudflare Pages
- Any web server (nginx, Apache, etc.)

### Option 3: Local Testing

```bash
# Extract the archive
tar -xzf portfolio-website-v1.0.0-build.tar.gz
# or
unzip portfolio-website-v1.0.0-build.zip

# Serve locally
npx serve out -p 3000

# Visit http://localhost:3000
```
