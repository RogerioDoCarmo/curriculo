## Visão Geral

Projeto educacional desenvolvido durante graduação em Ciência da Computação para explorar e dominar os fundamentos do desenvolvimento Android. Implementa diversos padrões e componentes essenciais da plataforma Android.

## Características Principais

### Componentes Android

- Activities e Fragments
- RecyclerView com ViewHolder pattern
- Intents e navegação entre telas
- Services e BroadcastReceivers
- Content Providers

### Persistência de Dados

- SQLite database local
- SharedPreferences para configurações
- Room Database (Android Architecture Components)
- Operações CRUD completas

### Interface do Usuário

- Material Design guidelines
- Layouts responsivos (ConstraintLayout)
- Animações e transições
- Temas claro e escuro
- Suporte a diferentes tamanhos de tela

### Arquitetura

- MVVM (Model-View-ViewModel)
- LiveData e ViewModel
- Repository pattern
- Dependency Injection básico

## Aprendizados

- Fundamentos sólidos de desenvolvimento Android
- Ciclo de vida de Activities e Fragments
- Padrões de arquitetura Android
- Boas práticas de UI/UX mobile
- Debugging e profiling de aplicações Android

## Impacto Educacional

- Base para projetos Android mais complexos
- Referência para estudos de desenvolvimento mobile
- Demonstração de compreensão de conceitos fundamentais
- Portfólio inicial de desenvolvimento Android
